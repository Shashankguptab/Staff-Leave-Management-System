# Staff Leave Management System (SLMS)

A full-stack web application built using **Python** and **Django** that helps organizations manage staff leave requests efficiently. 
The system allows employees to apply for leave and enables administrators to manage staff records and approve or reject leave requests.

## Live Demo

Live Website: https://shashank580.pythonanywhere.com/

https://github.com/Shashankguptab/Staff-Leave-Management-System

## Features

### Admin

* Secure Admin Login
* Add New Staff
* Edit Staff Details
* Delete Staff Records
* View Leave Applications
* Approve or Reject Leave Requests

### Staff

* User Registration
* Secure Login & Logout
* Apply for Leave
* View Leave History
* Track Leave Status

# Installation

### Clone the repository

```bash
git clone https://github.com/Shashankguptab/Staff-Leave-Management-System.git
```

### Go to the project directory

```bash
cd Staff-Leave-Management-System
```

### Create a virtual environment

**Windows**

```bash
python -m venv myenv
myenv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Apply migrations

```bash
python manage.py migrate
```

### Create a superuser

```bash
python manage.py createsuperuser
```

### Run the project

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
